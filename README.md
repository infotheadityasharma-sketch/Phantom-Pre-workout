# Phantom Pre-Workout

A responsive static marketing site for Phantom Pre-Workout.

## Run locally

Open `index.html` in a browser, or serve the directory with any static-file server.

## Project structure

- `index.html` — homepage
- `product.html`, `about.html`, `account.html`, `checkout.html` — supporting pages
- `styles.css` — site styles
- `cart.js` and `main.js` — cart and page interactions
- Image assets — product and visual assets

## Publish to GitHub Pages

1. Create an empty GitHub repository.
2. Push this project to its `main` branch.
3. In the repository, go to **Settings → Pages** and deploy from the `main` branch, using the repository root (`/`).
